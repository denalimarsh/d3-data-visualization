This is a collection of some data visualizations in d3.js I've worked on, featuring several visualizations about the worldwide refugee crisis, the java programming language, and tree maintenance in the city of San Francisco - as well as a couple miscellaneous side projects. I'm trained academically in tableau and d3v3/v4 through the University of San Francisco and have experience as a front-end developer integrating d3 into three.js for complex d3 visualizations in WebVR as a software engineer at Flow Inc.

Worldwide Refugee Crisis

Refugee Project Website

A series of visualizations aimed at highlighting the worldwide refugee crisis, giving meaning to the lives behind the statistics. Visualizations include a heatmap, an interactive geospatial choropleth, and a sankey diagram.

Java 8 API Hierarchy

Java 8 API Website

A look into Java's API hierarchy through several more complex D3.js visualizations, including a circular dendrogram, a traditional node link diagram, and two proprotional area maps (treemap and circle packing).

San Francisco Tree Maintenance

SF Tree Maintenance Website

A live, interactive geospatial map of reported tree maitenance cases within San Francisco, showing details on demand and individual case resolution.

Miscellaneous Visualizations

Miscellaneous Website

A small collection of miscellaneous visualizations, including a responsive bar chart, a geojson visualization of worldwide terrorism plotted by latitutde/longitude, and two visualizations on San Francisco's safety by neighborhood (bubble and bar chart).
